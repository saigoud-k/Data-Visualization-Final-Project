Medication Fulfillment Data Visualization
Project Overview

This project explores medication fulfillment data to understand prescription patterns, pharmacy activity, and medication supply trends. Using Python and data visualization techniques, the analysis highlights key insights such as the most prescribed medications, pharmacy distribution, monthly prescription trends, and medication supply duration.

The goal of this project is to demonstrate how data visualization can help identify patterns and improve decision-making in healthcare operations.

Dataset

The dataset used in this project is Medication Fulfillment data, which includes information such as:

Pharmacy Name

Drug Name

Dispense Quantity

Dispense Date

Days of Supply

This data helps analyze how prescriptions are distributed across pharmacies and medications.

Visualizations Created
1. Pharmacy vs Drug Heatmap

A heatmap showing the relationship between top pharmacies and the most frequently prescribed medications.
This helps identify which pharmacies handle the highest volume of specific drugs.

2. Top Medications by Prescription Count

A horizontal bar chart highlighting the most prescribed medications in the dataset.

Insights:

Identifies the medications with the highest demand.

Helps understand prescription distribution.

3. Monthly Prescription Trend

A time-series line chart showing how prescriptions change month by month.

Insights:

Detects seasonal patterns

Tracks prescription growth or decline over time

4. Distribution of Medication Supply Duration

A histogram with a density curve showing how many days of supply patients typically receive.

Insights:

Understands common prescription durations

Detects unusual supply patterns
