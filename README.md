# Medication Fulfillment Data Visualization

## Project Overview
This project analyzes medication fulfillment data to understand prescription patterns, pharmacy activity, and medication supply trends. Using Python and data visualization techniques, the project highlights key insights about the most prescribed medications, pharmacy distribution, monthly prescription trends, and medication supply duration.

The purpose of this project is to demonstrate how data visualization can help identify patterns and support better decision-making in healthcare data.

## Dataset
The dataset contains medication fulfillment information including:

- Pharmacy Name
- Drug Name
- Dispense Quantity
- Dispense Date
- Days of Supply

This information helps analyze prescription distribution across different pharmacies and medications.

## Visualizations

1. **Pharmacy vs Drug Heatmap**  
   Shows the relationship between pharmacies and frequently prescribed medications.

2. **Top Medications by Prescription Count**  
   A bar chart displaying the medications with the highest prescription counts.

3. **Monthly Prescription Trend**  
   A line chart showing prescription activity over time.

4. **Distribution of Medication Supply Duration**  
   A histogram showing the distribution of days of medication supply.
